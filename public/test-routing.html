Troubleshooting

If the links above result in 404 errors, check the following:

  1. Make sure your server is configured to redirect all requests to index.html
  2. For Apache, check if .htaccess is being processed (requires AllowOverride All)
  3. For Nginx, check if the try_files directive is included in your server block
  4. For static hosting services, make sure they support SPA routing
